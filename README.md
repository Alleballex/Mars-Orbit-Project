# Mars-Orbit-Project
This project considers a small satellite orbiting Mars at a low altitude gathering science data. However, this small satellite needs to transfer this data to a larger mother satellite at a higher altitude. Further, to keep the batteries from draining all the way, periodically the satellite must point its solar panels at the sun to recharge. 

The project considers nummerical time integration, defining different reference frames and a non-linear control system.

The project outline and objectives were created by Dr. Prof. Arya Vishala, for a full overview and understanding of the project please read PDF.
